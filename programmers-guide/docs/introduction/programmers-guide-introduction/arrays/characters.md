# Characters

Characters are entered within a pair of APL quotes.  The surrounding APL quotes are not displayed on output.  The APL quote character itself must be entered as a pair of APL quotes.

## Examples
```apl
      'DYALOG APL'
DYALOG APL
```
```apl
      'I DON''T KNOW'
I DON'T KNOW
```
```apl
      '*'
*
```
