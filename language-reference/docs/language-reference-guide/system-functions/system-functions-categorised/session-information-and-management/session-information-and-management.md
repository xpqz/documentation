# Session Information and Management

| Name | Description |
| --- | ---  |
| `⎕AI` | Account Information |
| `⎕AN` | Account Name |
| `⎕CLEAR` | Clear workspace (WS) |
| `⎕CY` | Copy objects into active WS |
| `⎕DL` | Delay execution |
| `⎕LOAD` | Load a saved WS |
| `⎕OFF` | End the session |
| `⎕PATH` | Search Path |
| `⎕SAVE` | Save the active WS |
| `⎕TS` | Timestamp |
