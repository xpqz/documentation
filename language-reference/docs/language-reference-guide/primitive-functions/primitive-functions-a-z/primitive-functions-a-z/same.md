




<h1 class="heading"><span class="name">Same</span><span class="command">R←⊣YR←⊢Y</span></h1>

`Y` may be any array.


The result `R` is the argument `Y`.

#### Examples
```apl
      ⊣'abc' 1 2 3
 abc  1 2 3
```
```apl
      (⊢,⎕size) 'a'⎕nl 4 ⍝ left tine of fork meaning "it"
acc      572
and      492
ascan    740
ascana   716
at      1764
avl    17476

```
