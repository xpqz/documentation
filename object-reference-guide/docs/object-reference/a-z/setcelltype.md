Description


This method is used to change the type of a particular cell in a [Grid](./grid.md).


The argument to SetCellType is a  3-element vector as follows :

| `[1]` | Cell row | integer |
| --- | --- | ---  |
| `[2]` | Cell column | integer |
| `[3]` | Cell type | integer |
